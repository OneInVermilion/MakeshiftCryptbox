# MakeshiftCryptbox
A personal project, relating to ciphers and file editing

Contains:
1. A tool for ciphering/deciphering text using most popular methods, such as Ceasar's Cipher or Atbash (At the time of writing this: Finished)
2. A tool for LSB Text-to-Image steganography with flexible settings (At the time of writing this: Finished)
3. A tool for editing the hex values of bytes in a file (At the time of writing this: Finished, requires optimization)
4. A tool for editing the metadata, specifically image exif data, such as camera that took a photo and its gps coordinates (At the time of writing this: Unfinished)
